## 📌 Deskripsi
Aplikasi ini memungkinkan pengguna untuk mengelola catatan secara digital dengan fitur CRUD (Create, Read, Update, Delete). Sistem dibangun menggunakan arsitektur fullstack dengan frontend dan backend yang saling terintegrasi melalui REST API.

## ⚙️ Teknologi yang Digunakan
- Frontend: HTML, CSS, JavaScript
- Backend: Express.js (Node.js)
- Database: MySQL

## 🚀 Fitur
- Tambah catatan
- Lihat daftar catatan
- Edit catatan
- Hapus catatan

## 🛠️ Cara Menjalankan

### Backend
```bash
npm install
node index.js
````

### Frontend

Buka file `index.html` menggunakan browser

## 📄 Catatan

Pastikan database MySQL sudah dibuat dan dikonfigurasi sesuai dengan koneksi pada backend sebelum menjalankan aplikasi.
